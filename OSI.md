**OSI**

*OSI(Open Systems Interconnection) Model is a 7-step model on how data is transferred and how networks communicate.It follows from step 1,The physical layer which involves all the physical connections that are in a network including switches,wires and routers.Whenever a problem arises in the physical layer,it is important to recheck the wiring,run feedback loops.The second layer is the data link layer,which constitutes the foundational layer of communication between devices.It follows the data link protocols involving MAC(Media Access Control) address over the ethernet.The 3rd layer is the network layer which handles routing and transmission between networks using IP Protocol.The 4th layer is the transport layer which involves data transmission between devices using TCP(Transmission Control Protocol) and UDP(User Datagram Protocol).The 5th layer is the session layer which facilitates communication between devices by opening/closing sessions.The 6th layer is the presentation layer which configues data into an acceptable format by translation/encoding/compressing it.The last layer is the application layer which is the interface wherein we can view/manipulate information by interacting directly with software services.*
